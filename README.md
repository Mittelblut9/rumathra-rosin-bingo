# Rumathra Bingo

This is a bingo game for the german streamer <a href="//twitch.tv/rumathra">Rumathra</a>.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
